Shape Grammars implementation class

Class Details

Instructor: Elena Vazquez. Professor: Jose Duarte

During Fall 2018 and Fall 2019, I taught the Shape Grammars implementation module for the Shape Grammars class.

The two week module consisted in exploring implementation software for Shape Grammars. We looked at several options, and tried out the Sortal plugin for Grashopper. The students had to implement a set of rules from their final term project, and evaluate different strategies for parametric variations.
